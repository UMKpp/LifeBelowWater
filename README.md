# SeaScape - Life Below Water 🌊

An educational and interactive web platform dedicated to preserving and protecting marine ecosystems for a sustainable future. This project focuses on the United Nations Sustainable Development Goal (SDG) 14: Life Below Water.

## 🔗 Live Demo
[https://life-below-water-eight.vercel.app/](https://life-below-water-eight.vercel.app/)

## ✨ Features
* **Interactive Educational Content:** Deep dives into marine ecosystems, including Corals, Marine Mammals, and Sea Turtles.
* **Conservation Awareness:** Highlights the importance of marine habitats, the threats they face, and actionable conservation efforts (like Marine Protected Areas).
* **Responsive Design:** Fully mobile-optimized layout that looks beautiful on desktops, tablets, and smartphones.
* **Volunteer Integration:** Dedicated volunteer registration pages allowing users to actively contribute to the cause.
* **User Feedback & Profiles:** Built-in systems to capture user feedback and personalized user profiles.
* **Visual Sitemap:** A dynamic and responsive sitemap to navigate the platform easily.
* **Data Visualization:** Comprehensive tables outlining endangered species, their conservation status, and ecological impact.

## 🛠️ Technologies Used
* HTML5
* CSS3 (Flexbox, CSS Grid, Advanced Media Queries)
* Vanilla JavaScript
* Hosted on Vercel
